Time de desenvolvedores

Objetivo do software
O CardHub é um sistema  criado para gerenciar uma loja de cartas colecionáveis, permitindo o cadastro de clientes, controle de cartas em estoque e registro de vendas.

Funcionalidades do sistema
- Cadastro de clientes
- Listagem de clientes
- Cadastro de cartas
- Listagem de cartas
- Registro de vendas
- Consulta de vendas
- Integração com banco de dados MySQL

Estrutura do sistema
O projeto foi desenvolvido seguindo a arquitetura em camadas:

- Model: classes de entidades
- DAO: acesso ao banco de dados
- View: interfaces gráficas
